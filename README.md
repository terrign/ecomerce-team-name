# Setup instructions:
1. Install [Node.js](https://nodejs.org/), if you haven't. Check if Node is installed:
  ```sh
  node -v
  ```
2. Clone the repo:
```sh
git clone https://github.com/terrign/ecomerce-team-name.git
```
3. Install NPM packages:
```sh
npm install
```
4. Run project locally:
```sh
npm start
```

# Scripts:
